1. Palíndromo:

Palavra ou frase que permanece a mesma lida de trás para frente. Exemplo: 

char animal[10] = "arara";

animal[0] == animal[4];
animal[1] == animal[3];

Outro exemplo com palavra de tamanho par: 

char palavra[5] = "esse";

É palíndromo pois: 

palavra[0] == palavra[3];
palavra[1] == palavra[2];

2. Ordenação de vetores

Podemos colocar os valores de um vetor em ordem crescente: 

int vetor[5] = {9, 7, 3, 6, 8};

Ficando assim: 

vetor[5] = {3, 6, 7, 8, 9};

Para começar, podemos verificar qual elemento é o menor entre dois deles assim: 

int maior;

int vetor[5] = {9, 7, 3, 6, 8};

if( vetor[0] > vetor[1] ) {
  maior = vetor[0];
  vetor[0] = vetor[1];
  vetor[1] = maior;
}

Se repetirmos o processo até o tamanho do vetor, teremos o maior de todos na última posição: 

int maior;

int vetor[5] = {9, 7, 3, 6, 8};

for(int i = 0; i < 4; i++) {
  if( vetor[i] > vetor[i+1] ) {
    maior = vetor[i];
    vetor[i] = vetor[i+1];
    vetor[i+1] = maior;
  }
}

2.2. Ordenação por bolha

O algoritmo que garante que um elemento está na posição correta cada vez que passa por todos os elementos do vetor é conhecido como Bubble Sort (Ordenação por Bolha), a passada deve ser repetida até que todos os elementos estejam na ordem. 

int maior;

int vetor[5] = {9, 7, 3, 6, 8};

for(int elemento = 0; elemento < 5; elemento++) {
  for(int i = 0; i < 4; i++) {
    if( vetor[i] > vetor[i+1] ) {
      maior = vetor[i];
      vetor[i] = vetor[i+1];
      vetor[i+1] = maior;
    }
  }
 }

Um loop for executa comandos uma determinada quantidade de vezes: 

for(int i = 0; i < 3; i++) // executa 3 vezes: i: 0, 1, 2

for(int i = 0; i <= 3; i++) // executa 4 vezes: i: 0, 1, 2, 3

for(int i = 1; i <= 3; i++) // executa 3 vezes: i: 1, 2, 3
